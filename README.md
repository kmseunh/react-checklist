# react-checklist

## 설명

이 프로젝트는 React를 이용한 체크리스트 앱입니다.

&nbsp;

## 목표

- 체크리스트를 제공하고 체크리스트 항목을 효과적으로 관리합니다.
- 항목이 체크되면 화면에서 사라지도록 구현합니다.
- `styled-components`를 사용하여 컴포넌트를 스타일링합니다.
